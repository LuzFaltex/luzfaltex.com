Lapis Lazuli

Lapis Lazuli, otherwise known as just Lapis, is an internal project name.

Lapis Lazuli is a self-hostable, secure, E2E encrypted community-based communication platform, melding the ideas behind Discord and Matrix. This does not strive to be a replacement for either platform, but users from both communities may like to leverage a common set of features.

Lapis Lazuli is currently under development. The following features are planned:

  • Centralized chat service,
  • Compliance with some Matrix standards (will not be fully compatible),
  • Open client API to connect user-provided clients,
  • Open bot API to connect automated services,
  • E2E encrypted private messaging and groups chats
  • Secure community spaces with integrated text, voice, and content channels (markdown pages, calendar system, and more),
  • Granular role-based permissions system,
  • Zero trackers, zero ads, and only anonymous analytics (number of users in community, generalized activity levels)
